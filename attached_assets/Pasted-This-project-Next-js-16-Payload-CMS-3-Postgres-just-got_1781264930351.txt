Visual Editor for the home page. The code is complete and tested. Please set it up
so everything works and is stored on Replit:

CONTEXT — WHAT WAS ADDED
- /editor — a visual editor (admin-only): live iframe preview of the home page,
  section list sidebar (drag to reorder, add/duplicate/hide/delete), click any
  section in the preview to select it, click any text to edit it inline, device
  preview (desktop/tablet/mobile), AR/EN toggle, autosave + undo/redo.
- New Payload global "home-page" (src/globals/HomePage.ts): the home page is now a
  blocks array of sections. The old home-* globals are hidden from the admin nav but
  still seed the editor's first save (automatic migration in src/lib/homePage.ts).
- API: src/app/api/editor/home/route.ts (GET/POST, requires a logged-in CMS user).
- Front end renders sections via src/components/home/HomeRenderer.tsx.
